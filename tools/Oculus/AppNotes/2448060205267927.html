Notes for Beat saber modding
Modding on Beat Saber version before 1.12.1 will most likely not work. If you plan to go to another version be sure to folllow the BSMG Quest modding guide
Downgrading guide
1. Log in on oculus.com
2. Download the desired version (most likely 1.13.2) from below
3. Uninstall Beat Saber via Sidequest
4. install the APK you downgraded in step 1
You've finished
